# Codex → GitHub Copilot の系譜

## tagline

Codex モデルが Copilot を産み、名前だけ 2025 年に CLI として復活した歴史です。

<!-- ━━━━━━━━ 左ページ ━━━━━━━━ -->

## 何をしてくれるか

OpenAI が 2021 年に発表した Codex はコード生成に特化した AI モデルです。GitHub Copilot の初期エンジンとして採用され、エディタ補完の原点になりました。2023 年に Codex モデルは廃止されましたが、2025 年に「Codex」の名前が CLI ツールとして再利用されました。

## どこで出会うか

「Copilot の中身は何か」「Codex って廃止じゃなかったの」という疑問が出たときにこの系譜と出会います。VS Code（ブイエスコード）の補完ツールとして Copilot を使うと、その裏側の歴史として参照することがあります。

## メイン図

### 図の狙い

2021 年の Codex モデル誕生から 2025 年の Codex CLI 復活まで、名前と実体がどう変わったかを 1 本の時系列で示します。

### C. 概念図（figure_type: timeline）

- 中心に置く概念: Codex という名前の「第一の生涯」（モデル）と「第二の生涯」（CLI）
- 周辺の要素: 2021 Codex モデル / 2021 Copilot Preview / 2022 Copilot GA / 2023 Codex 廃止 / 2025 Codex CLI 復活
- 関係の描き方: 左から右への時系列矢印、廃止箇所で一度途切れてから復活の矢印

## 開発フローでの位置（必須）

1. モデル誕生 — 2021-08 OpenAI が Codex モデルを発表（GPT-3 のコード特化版）
2. サービス化 — 2021-06 Copilot Preview、2022-06 GA で Codex がエンジンに採用される
3. 統合と廃止 — 2023-03 Codex モデルが GPT-4 系に統合されて廃止される
4. Copilot 拡張 — 2024 年に Copilot Chat・Workspace が拡充、モデルに Claude や Gemini も追加
5. 名前復活 — 2025-04 Codex CLI として自律型ツールに再登場し、別の系譜が始まる
