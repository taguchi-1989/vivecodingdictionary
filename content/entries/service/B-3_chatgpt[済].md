# ChatGPT

## tagline

OpenAI が提供する AI 対話サービス。最も広く普及した AI チャットの入口です。

<!-- ━━━━━━━━ 左ページ ━━━━━━━━ -->

## 何をしてくれるか

自然言語で話しかけると、文章作成・調査・コード生成・画像生成などを返してくれます。chatgpt.com・Custom GPT・API の 3 つの入口があり、用途によって使い分けられます。

## どこで出会うか

「AI を試してみよう」と検索した人が最初にたどり着くサービスです。同じ「ChatGPT」でも無料・Plus・API では体験が異なるため、どの入口の話かを揃えると会話が噛み合います。

## メイン図

### 図の狙い

ChatGPT というブランドの下に、アプリ・カスタム GPT・API という 3 つの入口があることと、それを支えるモデル（GPT-5 系）の関係を 1 枚で示します。

### B. 登場シーン（figure_type: structure）

- シーン1: 一般ユーザーが chatgpt.com で日常の質問や資料作成 — 「この企画書、もっと読みやすくして」
- シーン2: 業務担当が Custom GPT で用途特化のアシスタントを使う — 「社内規定に沿った回答だけ返す GPT」
- シーン3: 開発者が API を通してアプリに組み込む — 「チャットボットとして自社サービスに入れたい」
- 並べる基準: 使う人の立場別（一般 / 業務 / 開発）

## 会話での使い方例

「社内ナレッジは Custom GPT に寄せて、雑談は素の ChatGPT で十分ですね。」

## この用語の見どころ

### 1. 役割

OpenAI の AI 対話サービス。文章・画像・コードを同じ窓口で扱えます。

### 2. うれしさ

無料から始められ、事例と解説が最も多く情報を探しやすいです。

### 3. 注意点

無料・Plus・Pro・API で使えるモデルと上限が違います。

### 4. どこで役立つか

文章の下書き、資料の要約、コードの修正、アイデア出し全般。

### 5. はじめに

chatgpt.com で無料アカウントを作り、文章を 1 往復してみます。

### 6. 深掘り先

GPT-5 系、Custom GPT、DALL·E、OpenAI API

## 開発フローでの位置（必須）

1. アイデア出し・調査 — chatgpt.com で壁打ちや素材収集
2. 文章・資料の下書き — チャットで草稿を作り、人が仕上げる
3. コード補助 — 小さなスクリプトや修正を依頼する
4. アプリ組み込み — API 経由で社内ツールや自社サービスに ChatGPT の応答を繋げる
5. 専用 GPT の運用 — Custom GPT で用途特化のアシスタントを整備・維持
