# Claude の料金プラン

<!-- ━━━━━━━━ 左ページ ━━━━━━━━ -->

## tagline

個人 4 段（Free〜Max 20x）と法人 2 段（Team・Enterprise）からなる Claude の利用プランです。

## 何をしてくれるか

使用量・用途・チーム規模に合わせてプランを選ぶことで、Claude の利用上限や優先度が変わります。無料の Free から始め、必要に応じて上位プランへ移行できます。

## どこで出会うか

claude.com にアクセスすると料金ページで 6 段階のプランが並んでいます。Claude Code（略称 CC）など重い使い方をしていて「もっと使いたい」と感じたとき、Max プランの選択肢が目に入ります。

## メイン図

### 図の狙い

個人 4 段と法人 2 段の相対的な位置づけを、一目で比較できるようにします。

### B. 登場シーン（figure_type: comparison）

- シーン1: 無料で試したいユーザー — Free プランで Claude.ai の基本機能を使う
- シーン2: 日常的に活用する個人ユーザー — Pro プランで上限緩和と優先アクセスを得る
- シーン3: Claude Code を長時間動かすユーザー — Max プランで大量の操作に耐える
- 並べる基準: 利用量・用途別（軽め / 普通 / ヘビー / 法人）

## 会話での使い方例

「CC を長時間回すなら Pro では足りなくて、Max に切り替えました。」

## この用語の見どころ

### 1. 役割

使用量・優先度・機能範囲の上限を段階的に設定するプラン体系です。

### 2. うれしさ

Free から始めて必要なときだけ上位プランに移れるため、固定費を抑えやすいです。

### 3. 注意点

料金と上限は変動しやすく、evaluation_date 以降に改定されることがあります。

### 4. どこで役立つか

Claude Code や重いタスクを日常的に使う場合、Max プランが選択肢になります。

### 5. はじめに

個人 4 段と法人 2 段の構成と、Pro と Max の相対的な立ち位置を把握します。

### 6. 深掘り先

Claude、Claude Code、API、Anthropic。

## 開発フローでの位置（必須）

1. 試用 — Free で基本機能を確認する
2. 個人活用 — Pro で上限を緩め、日常の文章・コード作業に使う
3. 重い開発 — Max で Claude Code の長時間稼働に対応する
4. 法人展開 — Team / Enterprise でチーム管理や個別契約を整える
