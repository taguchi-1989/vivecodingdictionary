# Microsoft Copilot

<!-- バイブコーディング図鑑 エントリー雛形 v2（2ページ見開き想定、iter 22 準拠） -->

## tagline

Windows・Edge・スマホから呼べる Microsoft の汎用 AI チャットです。旧 Bing Chat の後継です。

<!-- ━━━━━━━━ 左ページ ━━━━━━━━ -->

## 何をしてくれるか

テキスト質問応答・Web 検索・画像生成（DALL·E 3）・音声会話（Copilot Voice）など幅広い機能を無料から提供します。中身は OpenAI の GPT 系モデルを主力に採用しています。

## どこで出会うか

Windows 11 のタスクバー・Edge ブラウザのサイドパネル・iOS / Android アプリに標準搭載されています。Microsoft アカウントさえあれば追加インストール不要で使い始められます。

## メイン図

### 図の狙い

「Copilot という名前は複数あるが、B-15 は無償の汎用チャット版」という立ち位置を、兄弟製品との比較で示す。

### B. 登場シーン（figure_type: comparison）

- シーン1: Windows ユーザーがタスクバーから呼んで日常質問に使う
- シーン2: Edge でページを開いたまま、選択テキストを Copilot に渡して要約させる
- シーン3: スマホアプリで Copilot Vision を使い、カメラ越しに実物を質問する
- 並べる基準: 「どのデバイス・入口から使うか」という入口の違い

## 会話での使い方例

「Edge から Microsoft Copilot を呼んで、開いている表をその場で要約させました。」

## この用語の見どころ

### 1. 役割

Windows や Edge に同梱された汎用 AI チャットです。

### 2. うれしさ

追加費用なしで画像生成や Web 検索付き AI が使えます。

### 3. 注意点

業務アプリ連携版（B-16）やコード補完版（B-5）とは別製品です。

### 4. どこで役立つか

日常的な調べ物や文章下書きの補助に向いています。

### 5. はじめに

「汎用チャット版 Copilot = B-15」とブランド整理から入ると混乱が減ります。

### 6. 深掘り先

Microsoft 365 Copilot、GitHub Copilot、Edge Copilot

## 開発フローでの位置（必須）

1. アイデア整理 — Copilot チャットで要件をざっくり言語化する
2. 調査・要約 — Edge サイドパネルで開いたページを即時要約させる
3. 文章・画像生成 — レポート下書きや挿絵を DALL·E 3 で作る
4. 確認・共有 — 生成結果をそのままコピーして会議資料に貼る

## コミュニティ補完メモ

- B-15（本エントリ）と B-16 Microsoft 365 Copilot の住み分け：B-15 は無償の汎用チャット版（旧 Bing Chat 系）。B-16 は Word・Excel・Teams などの Microsoft 365 アプリに統合された有料版。「Copilot」という名前は共通だが機能・料金・対象ユーザーが異なる
- B-5 GitHub Copilot との住み分け：B-5 はコードエディタ内の補完特化。B-15 はコード以外の日常タスク全般
- B-17 Edge Copilot との住み分け：B-17 は Edge ブラウザに特化した入口。B-15 はその上位概念に当たる汎用サービス。B-17 で呼び出した場合も実態は B-15 と同じモデルが動く
- C-8 Microsoft との関係：C-8 は企業・会社としての Microsoft のエントリ。B-15 はその AI サービス群の 1 製品
