# ElevenLabs

## tagline

テキストを自然な音声に変換する AI 読み上げサービスです。日本語を含む 30 言語以上と声のクローンに対応しています。

<!-- ━━━━━━━━ 左ページ ━━━━━━━━ -->

## 何をしてくれるか

テキストを入力すると、感情表現が豊かな音声を生成します。数秒の音声サンプルから声をクローンできる「Voice Cloning」機能も持ち、動画ナレーションやオーディオブック制作を一人で完結させられます。

## どこで出会うか

動画クリエイターやゲーム開発者が NPC の音声を量産する場面で名前を見かけます。バイブコーディングでは LLM の応答を音声化する MCP サーバーや音声エージェントの出力先として組み込まれます。

## メイン図

### 図の狙い

テキスト入力から多様な用途の音声が生まれる「使われ方の広がり」を示す。

### B. 登場シーン（figure_type: structure）

- シーン1: 動画クリエイターがナレーション用に自分の声をクローンして活用
- シーン2: 開発者がゲーム NPC の台詞テキストを一括で音声化
- シーン3: AI エージェントが LLM の応答をリアルタイム音声として返す
- 並べる基準: 職種と使いどころの違い（非技術 → 技術）

## 会話での使い方例

## この用語の見どころ

### 1. 役割

テキストを感情豊かな音声へ変換するクラウドサービスです。

### 2. うれしさ

数秒のサンプルから声をクローンし、多言語ナレーションを量産できます。

### 3. 注意点

他人の声を無断でクローンすることは規約違反で、本人確認手続きがあります。

### 4. どこで役立つか

音声エージェント・動画制作・ゲーム NPC など音声が必要な開発全般に役立ちます。

### 5. はじめに

Web の Speech Synthesis を試し、API キーで LLM と連携する流れを把握します。

### 6. 深掘り先

Whisper、音声エージェント、MCP

## 開発フローでの位置（必須）

1. テキスト準備 — LLM に応答を生成させ、読み上げ用テキストを用意します
2. API 連携 — ElevenLabs の API キーを取得して呼び出しコードを実装します
3. 声の選択 — プリセット音声か Voice Cloning で声を用意します
4. 音声出力 — 生成した音声ファイルをアプリや動画に組み込みます
5. 倫理確認 — クローン対象の本人同意と利用規約の遵守を確認します

## コミュニティ補完メモ

- Whisper（D-71）との住み分け：Whisper は音声→テキストの文字起こし。ElevenLabs はテキスト→音声の合成。方向が逆で補い合う関係
- Suno（B-60）との住み分け：Suno は音楽生成（歌・BGM）。ElevenLabs はナレーション・会話音声に特化
- Dictation（G-3）との住み分け：Dictation は入力補助ツール。ElevenLabs はサービス側の音声出力
- VLM（J-15）との住み分け：VLM は視覚理解モデル。音声エージェント構成時に LLM・VLM・ElevenLabs を組み合わせる文脈で関連
