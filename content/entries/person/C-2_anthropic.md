# Anthropic

## tagline

Claude を作り、最先端モデルを提供する AI 企業です。三大 AI 企業の一つです。

<!-- ━━━━━━━━ 左ページ ━━━━━━━━ -->

## ひとことで

Anthropic は、Claude を作っている AI 企業です。2021 年に OpenAI の研究リーダーだった人物たちが独立して設立しました。現在は OpenAI・Google DeepMind と並ぶ三大 AI 企業の一角です。

## 何をしてくれるか

- 言語モデルの開発と提供（Claude シリーズ）
- 製品提供（Claude.ai、Claude Code、Claude API、Amazon Bedrock 経由の提供など）
- AI 安全性の研究発信（Responsible Scaling Policy、Constitutional AI、解釈可能性の論文など）

最先端モデルの発表を継続し、三大 AI 企業の先頭集団に居続けている点が、日々の体感としての強みです。研究と製品化が並行していて、研究の主張が次のモデル・次の機能に直接反映されやすい会社でもあります。

## バイブコーディングでの位置づけ

本書で毎日触る Claude・Claude Code の「作っている本丸」です。Anthropic の方針が、次のモデルの挙動・料金体系・提供機能に直結します。

新モデルの発表ペース、ティアの刻み方、Claude Code の進化方向を追いかけていると、現場の道具のロードマップがそのまま見えるような感覚になります。「Claude が変わった」の背景には、常に Anthropic の判断があります。
