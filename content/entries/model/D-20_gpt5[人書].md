# GPT-5 系

## tagline

OpenAI の 2025 年フラッグシップ世代です。ChatGPT と API の両方を通じて使えます。

<!-- ━━━━━━━━ 左ページ ━━━━━━━━ -->

## 何をしてくれるか

テキスト生成・画像理解・コード生成を担うモデル系統です。GPT-4 系からの進化として、推論能力の向上とマルチモーダル（複数の入力形式への対応）強化が主な特徴とされます。

## どこで出会うか

ChatGPT のチャット画面と API 経由の開発の両方で名前を目にします。API 呼び出しコードでモデル名を指定するときや、利用プランの説明画面で識別子が現れます。

## メイン図

### 図の狙い

GPT-3 系から GPT-4 系、GPT-5 系への世代の流れを 1 本の矢印で示し、reasoning（推論）系との分岐も添えます。

### A. Before / After＋時系列（figure_type: timeline）

- GPT-3 系（2020〜）: 対話 AI の実用期への入口
- GPT-4 系（2023〜）: マルチモーダルと高度な長文処理が定着
- GPT-5 系（2025〜）: 推論能力・速度・マルチモーダルをさらに押し上げた現行フラッグシップ
- reasoning 系（o1 / o3）: GPT-5 系とは別系統。段階的思考に特化した派生ライン

## 会話での使い方例

「API でモデル名を指定するとき、GPT-4 系か GPT-5 系かで料金と速度が変わります。」

## この用語の見どころ

### 1. 役割

OpenAI の主力モデル系統で、ChatGPT と API の中心を担います。

### 2. うれしさ

GPT-4 系より推論・速度が向上し、同じ用途でより良い結果が出ることがあります。

### 3. 注意点

モデル名・料金・提供状況は時変情報のため、公式ページの再確認が必要です。

### 4. どこで役立つか

コード生成、長文要約、画像を含む問い合わせなど幅広いタスクに使えます。

### 5. はじめに

ChatGPT と API で使うモデルが同じ系統だと知ることが出発点です。

### 6. 深掘り先

GPT-4 系、o1 / o3 系、OpenAI API

## 開発フローでの位置（必須）

1. タスクを確認する — 対話・コード生成・画像理解など用途を絞る
2. モデルを選ぶ — GPT-5 系か reasoning 系（o1 / o3）かを判断する
3. 呼び出し方を決める — ChatGPT 画面か API 経由かを選択する
4. 実行と評価 — 出力の品質・速度・料金を確認し、必要なら変更する
