# o1 系

## tagline

答えを出す前に内部で考え込む、OpenAI の推論特化モデル系です。

<!-- ━━━━━━━━ 左ページ ━━━━━━━━ -->

## 何をしてくれるか

回答の前に内部で段階的な推論（Reasoning）を重ねるモデル系です。数学・論理・コード最適化など一発で答えが出にくい問題に向きます。o1・o1-mini・o1-pro の各バリアントがあります。

## どこで出会うか

ChatGPT（B-3）のモデル選択画面や、API（エーピーアイ）でモデル名を指定するときに目にします。GPT-5 系（D-20）より応答が遅めで、「なぜ時間がかかるのか」が最初のつまずきになりがちです。

## メイン図

### 図の狙い

GPT-5 系（速い・汎用）と o1 系（遅い・深く考える）を横並びに比較し、どちらをどの場面で選ぶかを人物の吹き出しで示します。

### A. 比較（figure_type: comparison）

- GPT-5 系: 汎用・高速 → 日常の文章生成・対話向き
- o1 系: 推論特化・じっくり → 数学・論理・コード最適化向き
  - o1-preview（2024-09、先行公開）
  - o1（量産版、o1-preview より速く安定）
  - o1-mini（軽量・低コスト版）
  - o1-pro（最上位、精度最重視）

## 会話での使い方例

「難しいロジックは o1 系に投げると、時間はかかりますが精度が変わりますよ。」

## この用語の見どころ

### 1. 役割

答えを出す前に内部推論を重ねる、OpenAI の推論特化系です。

### 2. うれしさ

論理・数学・最適化で通常モデルより精度が上がることがあります。

### 3. 注意点

内部推論のぶん応答が遅く、料金も高くなりがちです。

### 4. どこで役立つか

一発で答えが出にくい問題、バグ特定、数式の検証。

### 5. はじめに

o1・o1-mini・o1-pro の 3 バリアントと用途の整理から始めます。

### 6. 深掘り先

o3 系、Extended Thinking、OpenAI API

## 開発フローでの位置（必須）

1. タスクの難易度を見る — 論理・数学・深いコード理解かどうか判断する
2. モデルを選ぶ — 汎用なら GPT-5 系、推論が要るなら o1 系を選択する
3. バリアントを絞る — コスト重視なら o1-mini、精度重視なら o1-pro
4. 実行と評価 — 応答時間・品質・料金を確認し、必要なら切り替える
