# o3 系

## tagline

o1 系の後継にあたる推論特化モデル系です。内部推論をさらに深く回します。

<!-- ━━━━━━━━ 左ページ ━━━━━━━━ -->

## 何をしてくれるか

回答前に内部で段階的な思考（Reasoning）をより長く・深く重ねるモデル系です。ARC-AGI など難問ベンチマークで突出した成績を示し、数学・論理・コード最適化の難問に強みを発揮します。o3-mini・o3・o3-pro の 3 バリアントがあり、コストと精度の組み合わせで選べます。

## どこで出会うか

ChatGPT（B-3）のモデル選択や API（エーピーアイ）でモデル名を指定するときに現れます。o1 系より応答が遅く料金が高くなる場面があり、「どこまで深い推論が必要か」の判断が使いこなしの核です。

## メイン図

### 図の狙い

o1 系から o3 系への進化を時系列で示し、推論の深さが世代ごとに伸びていることを 1 枚で掴んでもらいます。

### A. 時系列（figure_type: timeline）

- o1（2024-09）: 推論特化の第 1 世代
- o3-mini（2025-01）: 軽量・低コストの o3 系先行版
- o3（2025-04）: 推論を深化させた主力版
- o3-pro（2025 年内）: 精度最重視の上位版

## 会話での使い方例

「この数学問題は o3 系に投げると、時間はかかりますが精度が段違いです。」

## この用語の見どころ

### 1. 役割

o1 系の後継として推論をさらに深化させた OpenAI のモデル系です。

### 2. うれしさ

難問ベンチマークで突出し、論理・数学の精度が一段上がることがあります。

### 3. 注意点

内部推論が深いぶんコストと遅延が大きく、用途を選びます。

### 4. どこで役立つか

数学・論理の難問、バグの根本原因特定、高精度なコード最適化。

### 5. はじめに

o3-mini・o3・o3-pro の 3 バリアントと、o1 系との違いを押さえます。

### 6. 深掘り先

o1 系、ARC-AGI ベンチマーク、OpenAI API

## 開発フローでの位置（必須）

1. タスクの難易度を見る — 一発で答えが出にくい論理・数学問題かを判断する
2. o1 系か o3 系かを選ぶ — さらに深い推論が要るなら o3 系を選択する
3. バリアントを絞る — コスト重視なら o3-mini、精度重視なら o3-pro
4. 実行と評価 — 応答時間・精度・料金を確認し、必要なら切り替える
