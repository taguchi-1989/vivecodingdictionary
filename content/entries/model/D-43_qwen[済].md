# Qwen 系

## tagline

Alibaba 発の多言語対応オープンモデル群です。中国語に特に強みがあります。

<!-- ━━━━━━━━ 左ページ ━━━━━━━━ -->

## 何をしてくれるか

Alibaba が開発・公開するテキスト生成モデル群です。Hugging Face からウェイトをダウンロードして手元で動かせます。コード・画像を扱う派生版も揃い多言語対応に優れています。

## どこで出会うか

ローカル LLM の選択肢やアジア言語タスクを調べる場面で名前が出ます。Qwen 2.5・Qwen 3 は Hugging Face のダウンロード数上位に入り、オープンモデル比較記事でよく取り上げられます。

## メイン図

### 図の狙い

Qwen 1.x から Qwen 3 系への世代推移と、Coder・VL などの派生ラインを 1 枚で把握してもらいます。

### A. 時系列タイムライン（figure_type: timeline）

- Qwen 1.x（2023）: 初公開。中国語・英語の二言語に特化した初世代
- Qwen 2（2024）: 多言語対応を強化。Apache 2.0 で公開されたモデルを含む
- Qwen 2.5（2024 後半）: Coder・VL（ビジュアル言語）・Math など派生版を拡充
- Qwen 3（2025）: シンキングモード搭載。MoE（Mixture of Experts）構造も採用

## 会話での使い方例

「Qwen 2.5 は多言語対応が比較的厚く、アジア言語タスクで試しやすいですよ。」

## この用語の見どころ

### 1. 役割

Alibaba 製のオープンウェイトモデル群。手元で動かせます。

### 2. うれしさ

中国語・日本語など多言語に強く、無償で利用できる版があります。

### 3. 注意点

ライセンスが版ごとに異なり、Apache 2.0 と独自ライセンスが混在します。

### 4. どこで役立つか

アジア言語タスクや、API コスト不要のローカル推論に向きます。

### 5. はじめに

Qwen 2.5 が安定した実績版、Qwen 3 が最新世代という世代感が要点です。

### 6. 深掘り先

Ollama、ファインチューニング、MoE

## 開発フローでの位置（必須）

1. モデルを選ぶ — 用途と GPU メモリに合わせて Qwen 3 / 2.5 の版を選ぶ
2. ウェイトを取得 — Hugging Face または qwen.ai からダウンロードする
3. ローカルで実行 — Ollama などで起動して日本語・中国語応答を確認する
4. 派生版を検討 — コード生成なら Qwen-Coder、画像入力なら Qwen-VL を試す
5. アプリに組み込む — API サーバー化して他ツールから呼び出す

## コミュニティ補完メモ

- Llama（D-40）との住み分け：Llama は Meta 製でオープンモデルの最大参照系統。Qwen は Alibaba 製で多言語（特に中国語・日本語）への強みが差別化点。本エントリでは「アジア言語タスクに強い別系統」として位置づけ
- Mistral（D-41）との住み分け：Mistral は欧州発・Apache 2.0 主体。Qwen はライセンス混在が注意点として際立つ
- DeepSeek（D-46/D-47）との住み分け：DeepSeek も中国発のオープンモデルだが、推論特化（R1）やコスト競争力（V3）が焦点。Qwen は多言語・派生の豊富さが要点
- Qwen 3 のシンキングモード（推論モード）詳細は D-47 DeepSeek R1 などの推論特化エントリと参照関係。本エントリでは「Qwen 3 で搭載」と触れる程度に留める
