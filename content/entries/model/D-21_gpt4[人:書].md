# GPT-4 系

## tagline

ChatGPT を一気に実用へ押し上げた世代です。GPT-4o がマルチモーダル化の起点です。

<!-- ━━━━━━━━ 左ページ ━━━━━━━━ -->

## 何をしてくれるか

テキスト生成・コード生成・画像理解を担うモデル系統です。2023 年に登場して ChatGPT の実用水準を引き上げ、GPT-4 Turbo で高速化、GPT-4o（オー）でマルチモーダル（複数の入力形式への対応）が広がりました。

## どこで出会うか

ChatGPT（B-3）でモデルを選ぶときや、API（エーピーアイ）でモデル名を指定するときに名前が出ます。「gpt-4o」「gpt-4-turbo」などの識別子でログや請求にも現れます。

## メイン図

### 図の狙い

GPT-4 → GPT-4 Turbo → GPT-4o → GPT-4.1 の系譜を 1 本の矢印で示し、各版の主な特徴と位置づけを添えます。

### A. Before / After＋時系列（figure_type: timeline）

- GPT-3 系（〜2022 年）: 対話 AI の実験期
- GPT-4（2023 年 3 月）: 実用水準を引き上げた世代の起点
- GPT-4 Turbo（2023 年 11 月）: 高速化・低コスト化
- GPT-4o（2024 年 5 月）: マルチモーダル本格化の転換点
- GPT-4.1（2025 年）: API 向け長文処理と性能改善
- GPT-5 系（2025 年〜）: 次世代フラッグシップへ

## 会話での使い方例

「GPT-4o から画像も入力できるようになったので、API のモデル名を確認するのが大事です。」

## この用語の見どころ

### 1. 役割

ChatGPT と API の中核を担った世代で、AI 実用化の起点です。

### 2. うれしさ

GPT-4o でテキストと画像をまとめて扱えるようになりました。

### 3. 注意点

版によって料金・速度・機能が異なるため、識別子の確認が必要です。

### 4. どこで役立つか

コード生成、長文要約、画像を含む問い合わせに使えます。

### 5. はじめに

GPT-4o がマルチモーダルの起点で、GPT-4.1 が API 向け改善版です。

### 6. 深掘り先

GPT-5 系、o1 系、OpenAI API

## 開発フローでの位置（必須）
