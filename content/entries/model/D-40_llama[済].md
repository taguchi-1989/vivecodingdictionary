# Llama 系

## tagline

Meta が公開するオープンモデルの旗手。ウェイトを手元で動かせます。

<!-- ━━━━━━━━ 左ページ ━━━━━━━━ -->

## 何をしてくれるか

Meta が開発・公開するテキスト生成モデル群です。ウェイトをダウンロードしてローカルで動かせるため、クラウド API なしで推論できます。Llama 1（2023年）〜 Llama 4（2025年）と世代を重ねたオープンモデルの代表格です。

## どこで出会うか

ローカル LLM を試す場面や社内閉域での推論構築で出ます。Ollama・LM Studio がデフォルト対応しており、「Llama 3 を動かす」というチュートリアルに多く登場します。

## メイン図

### 図の狙い

Llama 1 から Llama 4 までの系譜を時系列で示し、「世代ごとに何が変わったか」を 1 枚で捉えてもらいます。

### A. 時系列タイムライン（figure_type: timeline）

- Llama 1（2023-02）: 初公開。研究者向けに限定配布。7B〜65B パラメータ
- Llama 2（2023-07）: 商用利用を原則解禁。チャット版（Chat）も同時公開
- Llama 3（2024-04）: 性能が大幅向上。8B・70B・405B 版。多言語対応が進む
- Llama 4（2025）: マルチモーダル対応。Scout・Maverick・Behemoth の 3 段階

## 会話での使い方例

「Llama 3 をローカルで動かすと、API コストなしでファインチューニングを試せますよ。」

## この用語の見どころ

### 1. 役割

Meta 製オープンウェイトモデル群。ローカルで動かせる代表格です。

### 2. うれしさ

クラウド不要でデータを外に出さず推論でき、改造も自由です。

### 3. 注意点

Meta 独自ライセンスで厳密な OSS でなく商用条件を要確認です。

### 4. どこで役立つか

社内データを外に出せない場面やコスト削減のローカル推論。

### 5. はじめに

Llama 2 商用解禁、Llama 3 性能飛躍という世代の流れが要点。

### 6. 深掘り先

Ollama、ファインチューニング、量子化（Quantization）

## 開発フローでの位置（必須）

1. モデルを選ぶ — Llama 3（8B/70B）など用途と GPU に合わせて版を選ぶ
2. ウェイトを取得 — Hugging Face または llama.com からダウンロード
3. ローカルで実行 — Ollama や LM Studio で起動して動作確認
4. 必要に応じて調整 — ファインチューニングや量子化で用途に合わせる
5. アプリに組み込む — API サーバー化して他ツールから呼び出す

## コミュニティ補完メモ

- Mistral（D-41）との住み分け：Mistral は欧州発の別系統オープンモデル。Llama は「Meta 製・最も広く参照される系統」として位置づけ、比較対象として名前を出す程度に留める
- DeepSeek（D-46/D-47）・Qwen（D-43）・Gemma（D-42）はそれぞれ別エントリ。本エントリは「オープンモデルの源流として最も影響力が大きい系統」という立場で書く
- オープンモデル全般のライセンス・コピーレフト論争は別エントリへ。本エントリでは「Meta 独自ライセンス ＝ 厳密な OSS ではない」とだけ注記する
