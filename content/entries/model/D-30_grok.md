# Grok 系

## tagline

xAI が開発する LLM（大規模言語モデル）の系譜です。X との連携と比較的緩い検閲方針が独自色です。

<!-- ━━━━━━━━ 左ページ ━━━━━━━━ -->

## 何をしてくれるか

文章生成・コード生成・画像理解を担う LLM です。Grok-1 から始まり Grok-3・Grok-4 へと世代が進んでいます。Grok-4 には Heavy と Fast の 2 タイプがあり、重さと速度の使い分けが可能です。

## どこで出会うか

X（旧 Twitter）のプレミアム会員向けに組み込まれており、X 上でチャット形式で呼び出せます。grok.com でも単体利用でき、X のリアルタイム投稿データを参照できる点が他の LLM との差別化になっています。

## メイン図

### 図の狙い

Grok-1 → Grok-2 → Grok-3 → Grok-4 の世代の流れを時系列で示し、X 連携という独自の強みがどの世代から始まったかを 1 枚で伝えます。

### A. タイムライン（figure_type: timeline）

- Grok-1（2023）: xAI が初公開したオープンウェイトモデル
- Grok-2（2024）: 画像理解を追加した主力世代
- Grok-3（2025 前半）: 推論能力を強化した世代
- Grok-4（2025 後半〜）: Heavy / Fast の 2 タイプ体制、現行の主力

各世代で X のリアルタイムデータ連携が拡充されていく様子を右向き矢印で示します。

## 会話での使い方例

「Grok は X のリアルタイム情報を参照できるので、速報性のある話題では便利ですよね。」

## 開発フローでの位置（必須）

1. モデルを選ぶ — Grok-4 Heavy（高精度）か Fast（高速）かを決める
2. アクセス経路を決める — X プレミアム経由か grok.com か API か
3. X データ活用の要否を判断 — リアルタイム参照が必要かどうか
4. 実行と評価 — 他の LLM と比較しながら用途を絞り込む
