# Claude 4 系

## tagline

Claude 第 4 世代。Opus・Sonnet・Haiku の 3 段階で 4.7 が現行の主力です。

<!-- ━━━━━━━━ 左ページ ━━━━━━━━ -->

## 何をしてくれるか

文章生成・コード生成・画像理解・ツール呼び出しを担います。Opus（重い仕事）／Sonnet（日常の主力）／Haiku（軽量・高速）の 3 段階で、Claude Code などエージェント用途の主力帯になりました。

## どこで出会うか

Claude Code の主力モデル帯です。4.5 が大きな転換点で、4.6・4.7 はその安定化と伸長。どの版・ティアかでコスト・速度・品質のバランスが変わるため、使用版の把握が請求や体感の変化に直結します。

## メイン図

### 図の狙い

4 系の版の流れ（4.0 → 4.1 → 4.5 → 4.6 → 4.7）を、時系列の矢印とティア構成を重ねて 1 枚に収めます。どこが転換点だったかも示します。

### A. Before / After＋時系列（figure_type: timeline）

- Before（4 系登場前）: Claude 3.5 Sonnet でコーディング適性が跳ねた世代
- 4 系のマイナー版タイムライン
  - 4.0：Claude 4 系の初版
  - 4.1：微修正
  - **4.5：大きな転換点（著者感触としてシンギュラリティを感じた版）**
  - 4.6：安定化と能力の伸長
  - 4.7：現行の主力
- After（現行 4.7）: Claude Code の日常運用に十分耐える主力モデル帯に

各版について、縦に Opus／Sonnet／Haiku の 3 帯を引いて、それぞれの版でどのティアが出ているかを点で示します。

## 会話での使い方例

「重い仕事は Opus、日常は Sonnet ですね、4.5 から空気が一段変わった印象です。」

## この用語の見どころ

### 1. 役割

Claude シリーズ第 4 世代です。3 段階の賢さでタスクと予算を分けます。

### 2. うれしさ

仕事の重さに合わせて最も賢い／中間／アンカーの 3 段階を選べます。

### 3. 注意点

版（4.0〜4.7）によって性能と価格が動くので、評価日を意識して読みます。

### 4. どこで役立つか

コーディング、長文読解、複雑な指示のあるタスク全般。

### 5. はじめに

Opus／Sonnet／Haiku の役割と、4.5 が転換点・現行は 4.7 の流れ。

### 6. 深掘り先

Extended Thinking、Claude 4.5 系の詳細、Claude Code での自動選択。

## 開発フローでの位置（必須）

1. タスクの重さを見積もる — 調査系／実装系／反復系
2. 段階を選ぶ — 最も賢い（Opus）／中間（Sonnet）／アンカー（Haiku）
3. 実行 — Claude Code、Claude.ai、API のいずれか経由
4. 結果と費用を評価 — 必要なら一段上／下に切り替える
