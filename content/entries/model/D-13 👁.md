## tagline

Claude 4 世代の 4.5 系。コーディング能力と長文処理が大きく伸びた、2025 年の主力系統です。

<!-- ━━━━━━━━ 左ページ ━━━━━━━━ -->

## 何をしてくれるか

Claude 4 系の中で 4.5 の付番を持つモデル群です。Sonnet（ソネット）4.5 が先行し、続いて Haiku（ハイク）4.5 が登場しました。コーディング補助と長文処理が前世代より向上したと評されます。

## どこで出会うか

Claude Code や Claude.ai でモデル選択画面に表示されます。「claude-sonnet-4-5」「claude-haiku-4-5」のモデル ID で API から呼び出す場面でも目にします。

## メイン図

### 図の狙い

4.5 世代の 2 モデル（Sonnet 4.5 → Haiku 4.5）が時系列でどう展開したかを示し、4 系の他版との立ち位置を掴んでもらいます。

### A. タイムライン（figure_type: timeline）

- 横軸：2025 年中の登場順（Sonnet 4.5 → Haiku 4.5）
- 縦軸：ティア（Sonnet ＝ 中間 ／ Haiku ＝ 軽量・高速）
- 4 系全体（4.0〜4.7）のライン上に 4.5 系のドットを重ねる
- 前後版との改善点として「コーディング↑」「長文処理↑」のラベルを添える


## 会話での使い方例

「コーディング系のタスクは Sonnet 4.5 に投げると速いですよ。」

## この用語の見どころ

### 1. 役割

Claude 4 系の 4.5 付番を持つ Sonnet・Haiku の 2 モデルです。

### 2. うれしさ

コーディングと長文処理が前世代より向上したと評されます。

### 3. 注意点

性能・提供状況は時変情報で、評価日以降に変わることがあります。

### 4. どこで役立つか

コード生成・レビュー・長文要約など日常の AI 補助全般。

### 5. はじめに

Sonnet が中間、Haiku が軽量・高速という 2 ティアの役割を掴みます。

### 6. 深掘り先

D-12 Claude 4 系、D-11 Claude 3.5 系、Extended Thinking

## 開発フローでの位置（必須）

1. タスクを選ぶ — コーディング・要約など 4.5 世代が得意な作業を特定する
2. ティアを決める — 精度優先なら Sonnet 4.5、速度・コスト優先なら Haiku 4.5
3. モデル ID を指定 — API や Claude Code でモデル名を確認して呼び出す
4. 結果を評価 — 性能や費用を見て 4 系の他版への切り替えを検討する
