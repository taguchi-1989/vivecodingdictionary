# IQ Bench

<!-- ━━━━━━━━ 左ページ ━━━━━━━━ -->

## tagline

LLM の「パターン推論力」を測るベンチマーク群の総称です。知識暗記ではなく未知の図形や規則を読む力を問います。

## 何をしてくれるか

Raven's Progressive Matrices（行列推論テスト）風の課題で、LLM が図形の続きや抽象ルールを推測できるかを数値化します。知識ベンチで高得点のモデルでも、IQ 換算で人間平均前後に収まる傾向です。

## どこで出会うか

AI 研究者や業界メディアが「LLM はテスト勉強型」と議論する場面で登場します。Yann LeCun（C-56）が LLM の限界を語るときや、AGI（J-1）到達の議論で引かれます。

## メイン図

### 図の狙い

知識ベンチと IQ 系ベンチのスコア差を並べて、LLM の得意・不得意を一目で示す。

### B. 登場シーン（figure_type: comparison）

- シーン1: 知識ベンチ（GPQA など）で満点近いモデルが IQ 系では 80 前後にとどまる
- シーン2: Raven's Progressive Matrices 風の図形問題に LLM が答えようとする場面
- シーン3: AGI 議論で「人間の IQ 100 を超えたか」の指標として引用される
- 並べる基準: 知識型ベンチ vs 推論型ベンチのスコア対比

## 会話での使い方例

「IQ Bench では、知識ベンチで強いモデルでも人間平均レベルが多いそうですね。」

## この用語の見どころ

### 1. 役割

知識ではなく未知パターンの推論力を測る指標です。

### 2. うれしさ

LLM の「暗記以外の弱点」を可視化できます。

### 3. 注意点

単一の標準ベンチはなく、複数の IQ 系テストの総称です。

### 4. どこで役立つか

AGI 到達議論やモデル選定の比較材料になります。

### 5. はじめに

LLM が暗記型で推論が弱い点を起点にすると入りやすいです。

### 6. 深掘り先

AGI、ARC-AGI、GPQA

## 開発フローでの位置（必須）

1. モデル選定 — 候補モデルの知識ベンチと IQ 系ベンチを並べて比較します
2. 能力評価 — Raven's 風課題でパターン推論力のスコアを確認します
3. 弱点把握 — 高スコアモデルでも IQ 換算 70〜100 帯にとどまる箇所を記録します
4. 議論参照 — AGI 達成基準の議論で IQ Bench の結果を引用・共有します

## コミュニティ補完メモ

- GPQA（E-22）との住み分け: GPQA は専門知識問題の正答率を測る知識系ベンチ。IQ Bench は暗記ではなくパターン推論力を測る点で区別される
- Humanity's Last Exam（E-26）との住み分け: E-26 は知識の難易度上限を探るベンチ。IQ Bench は難易度より「推論の種類」の違いを測る
- ARC-AGI との関係: ARC-AGI は IQ 系ベンチの代表例として本エントリ内に含めて紹介
