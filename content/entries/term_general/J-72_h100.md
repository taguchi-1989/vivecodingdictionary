# H100


<!-- ━━━━━━━━ 左ページ ━━━━━━━━ -->

## tagline

NVIDIA Hopper 世代のデータセンター向け GPU（グラフィックス処理装置）。AI 学習・推論の物理的な土台です。


## 何をしてくれるか

大量の行列演算を並列処理し、LLM（大規模言語モデル）の学習と推論を高速化します。80GB の VRAM（J-70）と第 4 世代 NVLink で複数枚を束ね、数万枚規模のクラスタとして動きます。


## どこで出会うか

GPT-4 や Claude 3 などの LLM 学習ニュースで「H100 数万枚」という表現を目にします。AWS p5・GCP A3 など主要クラウドの GPU インスタンスで時間貸しされており、API の裏側でも稼働しています。


## メイン図

### 図の狙い

H100 が単体 GPU から大規模クラスタへと束ねられ、LLM 学習を支える仕組みを示す。

### C. 概念図（figure_type: structure）

- 中心に置く概念: H100 クラスタ
- 周辺の要素: 1 枚の H100 ボード、NVLink 接続、クラウドインスタンス、LLM 学習ジョブ
- 関係の描き方: H100 複数枚を NVLink で接続 → クラスタ → LLM 学習、という矢印の流れ


## 会話での使い方例

「GPT-5 は H100 数万枚で学習したらしいですね。」

## この用語の見どころ

### 1. 役割

AI 学習・推論を支えるデータセンター向け GPU です。

### 2. うれしさ

前世代 A100 比で演算速度が大幅に向上しました。

### 3. 注意点

1 枚 3〜5 万ドルで個人購入は現実的ではありません。

### 4. どこで役立つか

LLM 学習クラスタやクラウド GPU インスタンスで使われます。

### 5. はじめに

「1 万枚規模のクラスタ」という業界の桁感を押さえると読めます。

### 6. 深掘り先

GPU、VRAM、NVLink


## 開発フローでの位置（必須）

1. モデル設計 — 学習に必要な VRAM 量と H100 枚数を見積もります
2. クラスタ調達 — クラウドの GPU インスタンスや自社設備で H100 を確保します
3. 分散学習 — NVLink で束ねた H100 クラスタで並列学習ジョブを回します
4. 推論デプロイ — 学習済みモデルを H100 搭載インスタンスで API として公開します
