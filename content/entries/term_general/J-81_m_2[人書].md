# M.2

## tagline

基板に直挿しする小型ストレージ規格です。NVMe SSD の取り付けに広く使われます。

<!-- ━━━━━━━━ 左ページ ━━━━━━━━ -->

## 何をしてくれるか

M.2 はマザーボードのスロットに直接差し込む薄型の接続規格です。ケーブル不要で、NVMe（エヌブイエムイー）SSD や Wi-Fi カードの取り付けに使います。接続方式は SATA と NVMe があり、外見が同じでも速度に差が出ます。

## どこで出会うか

自作 PC やノート PC のスペック表で「M.2 スロット × 2」のように記載されています。ローカル LLM（大規模言語モデル）を動かす PC を組む場面で、NVMe M.2 SSD を選ぶか検討する際に目にします。

## メイン図

### 図の狙い

M.2 スロットへの差し込み形状と、SATA 接続と NVMe 接続の速度差を対比で示します。

### B. 登場シーン（figure_type: comparison）

- シーン1: NVMe M.2 SSD — モデル切替が高速、ローカル LLM 向け
- シーン2: SATA M.2 SSD — 低コストだが NVMe より読み書き遅い
- シーン3: Wi-Fi / Bluetooth カード — M.2 スロットに挿す無線カードも存在
- 並べる基準: 同じ形状でも接続方式・速度・用途が異なる点を対比


## 会話での使い方例

「NVMe の M.2 SSD に換装したら、ollama のモデル切替が一瞬で済むようになりました。」

## この用語の見どころ

### 1. 役割

マザーボードにケーブルなしでストレージや無線カードを接続する規格です。

### 2. うれしさ

NVMe 接続の SSD を選ぶと、ローカル LLM のモデル読み込みが高速になります。

### 3. 注意点

形状が同じでも SATA と NVMe では速度が大きく違うため、購入前に仕様確認が必要です。

### 4. どこで役立つか

ローカル LLM 用の自作 PC を組む際に SSD の接続方式を選ぶ場面で役立ちます。

### 5. はじめに

M.2 は形状規格、SATA・NVMe は接続方式の違いを把握するところから始めます。

### 6. 深掘り先

NVMe、SATA、PCIe


## 開発フローでの位置（必須）

1. PC 構成検討 — ローカル LLM 用に高速ストレージが要るか要件を整理します
2. M.2 スロット確認 — マザーボードのスロット数と対応規格を調べます
3. SSD 選定 — NVMe 接続の M.2 SSD を選ぶと読み書き速度が伸びます
4. 取り付け — スロットに差し込みネジ止めするだけでケーブル配線は不要
5. 動作確認 — OS や ollama でモデルロード速度の改善を確認します

## コミュニティ補完メモ

- J-80 SATA との住み分け：SATA は接続インターフェース規格（SATA I/II/III の速度体系）を説明する。M.2 は形状規格であり、M.2 スロットが SATA 接続と NVMe 接続の両方を取れる点が異なる
- J-79 SSD との住み分け：SSD はストレージデバイスの種別。M.2 はその取り付け形状の 1 つ。「M.2 形状の SSD には NVMe と SATA の 2 種類がある」という上下関係
- F-86 ollama との接点：ローカル LLM ランタイムとして ollama を使う際、モデルファイルを格納するストレージが NVMe M.2 SSD だとロード速度が大幅に向上する
- J-78 HDD との住み分け：HDD は磁気ディスク。M.2 は主に SSD 用の形状規格であり、HDD には M.2 形状は存在しない

## 備考

- M.2 のサイズ表記（例: 2280）は横 22mm × 縦 80mm を意味する。デスクトップでは 2280、薄型ノートでは 2242 や 2230 が使われることがある
- NVMe 接続でも世代（PCIe 3.0 / 4.0 / 5.0）によって速度差があるため、マザーボードと SSD の世代組み合わせも購入前に確認が推奨される
- M.2 Key（M キー / B キー / B+M キー）の違いによってスロットとの互換性が変わるケースがあるが、現行市場では M.2 NVMe SSD の多くが M キーのため通常意識不要
