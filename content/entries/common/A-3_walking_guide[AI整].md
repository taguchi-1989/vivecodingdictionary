## リード文

通読は不要です。気になる語から引いて、関連用語のピルでつないでください。letter 順と掲載順は別で、本書は「どこから入っても帰ってこれる」造りにしています。

## A〜J 地図と 3 ルート

A〜J の章を円環または横並びの地図にして、章ごとのテーマラベルを置きます。色違いの線で 3 ルートを重ねます。

- 章テーマ: A はじめに / B サービス / C 人・組織 / D モデル / E ベンチマーク / F 従来コード / G バイブ特有 / H 歴史 / I MCP / J 一般語彙
- ルート①索引引き: A-9 の索引から気になる語へ直接飛ぶ。最短経路
- ルート②業界マップ順: A→C→D→B→I の推奨順で、人・組織 → モデル → サービス → MCP の流れで全体像を掴む
- ルート③レベル別: A-5 の reader_level（1〜6）を参考に、自分の前提知識に合う語から拾う
