# J-14 LLM ponchi regeneration prompt

## Purpose

Generate a 2:1 book ponchi image for the LLM entry.

## Key idea

An LLM receives context/tokens, processes them through a layered model, and
returns the next text-like output. The image should show this as a clean
conceptual flow, not as a product screen.

## Base image prompt

Create a 2:1 landscape editorial spot illustration for a Japanese
AI/development glossary entry about LLMs. Canvas 1254x627, pure white
background.

Composition:
- Left: a reader figure sends an abstract input card into the model.
- Center: a large simple model box with a light grid/layer pattern inside,
  shown as an abstract language model, not a branded product.
- Bottom: small generic data tiles flow upward into the model as muted gray
  icons, not real websites or logos.
- Right: output cards flow out as abstract horizontal lines that suggest
  generated language without readable text.
- Use clean arrows, generous whitespace, and a clear left-to-right flow.

Character continuity:
- Use recurring Character A from `docs/ponchi_character_bible.md` for the
  reader/listener: anonymous adult woman, shoulder-length black hair, minimal
  dot eyes, white or very light gray top with a dark simple cardigan outline,
  no accessories, no patterned clothing.
- If AI assistance needs a character, use only recurring Character C as a small
  neutral pet robot. Do not create a new mascot or animal-like character.

Style:
- Clean simple editorial line illustration, not pencil sketch.
- Smooth uniform black lines, flat light gray fills, minimal shading.
- No hatching, scratchy hand-drawn texture, marker texture, or painterly look.

Color:
- Follow `docs/ponchi_image_generation_rules.md`.
- Use only `#FFFFFF`, `#F7F9FC`, `#1A1A1A`, `#6B7280`, `#EAF1FB`, `#D6E6FA`,
  `#8DB7E8`, `#3F7FD1`, `#123E82`.
- Use blue only for the active processing path through the model and output.

Text and logo rules:
- No readable text, no code snippets, no product UI screenshots, no logos, no
  trademarks, no brand colors, no watermarks.
- Do not write "LLM" inside the image.
- Do not generate official app icons, product icons, official marks, official
  mascots, brand-color substitutes, or real web/product UI.

Avoid:
- Robot mascot, brain illustration, colorful internet logos, dense diagram,
  speech bubbles, fake readable text, decorative blue sparkles.
