A-3 · FRONT · HOW TO FIND

語の探し方

引き方は 3 つ。どれを選んでも「気になる語 → 関連語のピル」でつながり、
数分で現場の会話に戻れる造りです。

①索引引き: 巻末の全件索引から、気になる語へ直接飛ぶ（最短ルート）
②業界マップ順: A → C → D → B → I（人 → モデル → サービス → MCP）で全体像をつかむ
③レベル別: A-5 の reader_level に合わせて、自分の段階の語から拾う
letter 順ID から位置で引く（F-2 など）
五十音順日本語名で引く（「コンテキスト」など）
アルファベット順略称で引く（GPT, MCP, RAG など）
語の一覧は 巻末の全件索引（全 354 語） にまとめました。ここでは「どう探すか」だけ示します。
