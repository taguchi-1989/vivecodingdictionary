# 後付け（back matter）— HTML たたき台

巻末側のページ（あとがき・本物索引など）の HTML 下絵を集約するディレクトリです。前付けの [drafts/front_section/](../front_section/) と対になります。

## 現在のファイル

| ファイル | 内容 |
|:--|:--|
| `afterword.html` | あとがき — この本がどう作られたかを著者の声で書く 1 見開き |

## まえがき・あとがきの役割分担

| 配置 | 担当ファイル | 内容 |
|:--|:--|:--|
| 巻頭 まえがき | [drafts/front_section/1_a1_preface.html](../front_section/1_a1_preface.html) | 本書を作った動機と、誰に向けた本かを書く。読者向けに「この本は何の本か」を伝える |
| **巻末 あとがき** | `afterword.html` | 本書をどう作ったか（多エージェント並列・対話口述・AI 共著）を書く。経緯と数値・反省をまとめる |

「この本を作った経緯」はまえがきよりあとがき側のほうが据わりが良い、という判断で巻末側へ。読者は本編を読んだあとに「どうやって書いたんだろう」と振り返る場所として読みます。

## 共通スタイル

`drafts/front_section/_common.css` を共有しています（spread レイアウト・チロム・タグ帯等）。トーンを揃えるため、別系統の CSS は作っていません。
