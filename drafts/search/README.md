# 検索 UI — たたき台

327 件のエントリを「ぱっと見て検索できる」UI の HTML プロトタイプです。

## 使い方

1. `python3 scripts/build_search_index.py` でインデックス JSON を生成（既に生成済み）
2. `drafts/search/index.html` をブラウザで開く
3. 検索窓に入力すると即座にフィルタ・サジェスト

## 機能

### 検索

- **タイトル**（漢字／カタカナ）
- **タイトル読み**（「コンテキスト」「ジット」など）
- **ID**（`B-3`, `MCP` など）
- **tagline**（説明文の中身）
- **関連用語**

優先順位は: 完全一致 > 前方一致 > 部分一致（タイトル）> ID 一致 > 説明文一致 > 関連語一致。

### フィルタ

横並びのピルで絞り込み：

- **章** — A〜J で 10 章
- **状態** — ready / needs_review / drafting / skeleton
- **レベル** — Lv 1〜6

複数を組み合わせ可能。

### 隣接概念サジェスト

検索結果の上位 5 件の `related_terms` をたどって、まだ表示していないエントリを別枠に「隣接概念のサジェスト」として表示。検索意図に近い周辺語が自然に見えます。

### キーボードショートカット

- `/` でフォーカス（検索窓へ）
- `Esc` でクリア

### クリック動作

- 結果カード内の「関連用語ピル」をクリック → そのキーワードで再検索

## 表示例

各カードに表示する情報：

| 要素 | 例 |
|:--|:--|
| ID バッジ | `B-3`（章別の色分け） |
| タイトル | ChatGPT |
| タイトル読み | チャットジーピーティー |
| Lv バッジ | Lv 1 |
| 状態ドット | 🟢 完成 / 🟠 レビュー待 / 🟤 書きかけ |
| tagline | 説明文（マッチ箇所をハイライト） |
| 関連用語ピル | 最大 6 個（クリックで再検索）|

