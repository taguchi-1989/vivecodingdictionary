## 0. 書くときの 1 分チェック

- [ ] `templates/entry_template.md` をコピーして開いている
- [ ] **タイトルは純粋名のみ**（`Context（コンテキスト）` のように括弧で読み・日本語訳を含めない。読みは下のスロットへ）

- [ ] **タイトルが略称・ヌメロニム（MCP / a11y / LLM 等）なら、`tagline` の冒頭に正式名称（または日本語展開）を入れる**（例: `Model Context Protocol の略。…` / `accessibility の略（a と y の間に 11 文字）。…` / `大規模言語モデルの略。…`）

## 2. 文字数と項目数の決定版ルール

**本節がすべての執筆・検証の単一真実点**。writing_spec.md・validate_entry.py・check_entry.py は本表を写す側。数値を変えるときは本表を先に動かす。

### 2-1. 定義の読み方

- **文字数**は空白・HTML コメント・Markdown 装飾（`**` `-` 等）を除いた実文字数
- **項目数**は「固定」「範囲」を明記。範囲の場合は両端含む
- **判定**列の記号:
  - ❌ = check_entry.py が **エラー**（執筆を止める、必ず直す）
  - ⚠️ = validate_entry.py が **警告**（推奨範囲、超えても書ける）
  - ℹ️ = 情報表示のみ（機械判定なし、目視）
- **推奨**と**許容**を分ける節は、推奨＝執筆時の目標、許容＝誌面が破綻しない上限

### 2-2. YAML フロントマター

| キー | 種別 | 値 | 判定 |
|---|---|---|---|
| id | 必須 | `{letter}-{num}` 形式（`F-1`, `D-12` 等） | ❌ |
| title | 必須 | 用語名（純粋名のみ。括弧書きの読み・展開は title_reading へ）。2〜20 字推奨 | ❌ |
| title_reading | 任意 | **読み専用**スロット。**英字略称・英字タイトルは英単語をカタカナで並べた読み**を入れる（例: `モデルコンテキストプロトコル` / `ラージランゲージモデル` / `アクセシビリティ` / `ギット` / `コンテキスト`）。略称音読み（えむしーぴー 等）や日本語訳（大規模言語モデル 等）は使わない。**正式名称は tagline 冒頭へ回す**。2〜30 字推奨 | ⚠️ |
| category | 必須 | service / model / term / tool / workflow / history / person / benchmark / common / mcp | ❌ |
| subtype | 任意 | language / framework / ai_assistant などの下位分類 | ℹ️ |
| experience_level | 推奨 | hands_on / partial / research_only | ⚠️ |

| page_layout | 必須 | `spread_v1` | ❌ |

| pricing_note | 時変のみ | none / paid / freemium | ⚠️ |

| status | 必須 | drafting / sample / needs_source / needs_review / ready / archived | ❌ |

### 2-3. 左ページ本文

| 箇所 | 必須 | 文字数（許容） | 文字数（推奨） | 文数 | 判定 |
|---|---|---|---|---|---|
| **タイトル**（YAML title 参照） | ○ | 2〜20 字 | 2〜12 字 | — | ⚠️ |
| **タイトル読み**（YAML title_reading 参照） | △任意 | 2〜30 字 | 3〜15 字 | 1 行 | ⚠️ |
| **tagline**（青帯） | ○ | **25〜60** | 30〜38（略称展開含む場合 35〜50） | 1〜2 文 | ❌ |
| 体験区分タグ | ○ | enum から | — | — | ℹ️ |
| 推奨読者レベルタグ | ○ | enum から | — | — | ℹ️ |
| **何をしてくれるか** | ○ | **60〜200** | 80〜150 | 2〜4 文 | ❌（下限・です/ます語尾必須） |
| **どこで出会うか** | ○ | **60〜200** | 80〜150 | 2〜4 文 | ❌（下限・です/ます語尾必須） |
| Before/After 図 | figure_type 次第 | キャプション各 10〜25 字 | — | — | ⚠️ |
| **擬人化ポンチ絵 caption** | ○ | **30〜80** | 40〜60 | 1〜2 文 | ⚠️ |
| **会話での使い方例** | ○ | **25〜50** | 30〜40 | 1 文 | ⚠️（範囲外） |

**tagline の禁止**: title と同じ単語を先頭に置く（`TypeScript は、TypeScript に型…`）。validator が検出。

**tagline の略称展開ルール（2026-04-28 追加）**:

- タイトルが略称・略字・ヌメロニム（MCP / LLM / a11y / i18n / GDPR など）の場合、tagline の **冒頭に正式名称または日本語展開を 1 句で入れる**。
  - 例（英字略称）: `Model Context Protocol の略。LLM とツール・データをつなぐ標準規格です。`
  - 例（ヌメロニム）: `accessibility の略（a と y の間に 11 文字）。Web を誰でも使える状態に保つ取り組みです。`
  - 例（日本語展開）: `大規模言語モデルの略。文章で訓練された AI の主流で、主要 3 サービスの中身です。`
- 形式の目安は **「{展開} の略。{1 文の定義}。」**。許容字数は 25〜60、推奨 35〜50（略称展開ぶん上限を緩めた）。
- **`title_reading` には正式名称を入れない**（読み専用）。展開は tagline 側に置く。
- 純粋名のタイトル（Context / Git / Claude 等）には適用しない。tagline は通常の推奨範囲 30〜38 字（許容 25〜45 字）で書く。

共通ルール: 各セル **1 文**、カンマ並列は 3 語まで、名詞止め可

**iter 22 検証で方針補強**: 許容 15-40 字を維持しつつ、**推奨 15-30 字**を目安にする。TypeScript 見本の実測（19-26 字）に合わせて、右ページ全体を詰めた状態に保つ。30 字超は validator の ⚠️ 警告対象。

|---|---|---|---|---|---|
| 1 | 役割 | **15〜40** | 15〜30 | 的 ＋ 十字（bullseye with crosshair） | ❌（許容） / ⚠️（推奨） |
| 2 | うれしさ | **15〜40** | 15〜30 | ハート | ❌ / ⚠️ |
| 3 | 注意点 | **15〜40** | 15〜30 | 警告三角 | ❌ / ⚠️ |
| 4 | どこで役立つか | **15〜40** | 15〜30 | 電球 | ❌ / ⚠️ |
| 5 | はじめに | **15〜40** | 15〜30 | 双葉の芽（Lucide sprout） | ❌ / ⚠️ |
| 6 | 深掘り先 | **15〜50** | 15〜35 | 文書 + 虫眼鏡（file-search） | ❌ / ⚠️ |

**禁止**: セル内箇条書き（カンマ列挙はギリギリ許容）、前セルと同じ文頭連発。

## 3. 文体ルール

- **です・ます調 必須**（CLAUDE.md 準拠）
- 結論から書く
- 読者を馬鹿にしない、強い断定を避ける、煽らない
- 絵文字は **私のコメント** の 4 種（★/✓/!/☺）以外 使わない
- tagline は title の言い換えでなく**一歩先の定義**
- 箇条書き連発・言い換え連続を避ける

## 6. 自動チェッカーの使い方

執筆後:

```bash
python drafts/prototypes/mockups/design_philosophy_v2/check_entry.py path/to/entry.md
```

全件監査:

```bash
python drafts/prototypes/mockups/design_philosophy_v2/check_entry.py --dir content/entries/
```

チェック項目:
- YAML 必須キー（id / title / category / figure_type / page_layout / evaluation_date）
- 必須節の存在
- 各節の文字数レンジ
- ですます語尾（本文節）
- tagline が title の言い換えでない
- 関連用語 3〜5 個
- 旧節名の互換警告

`status: archived` のエントリは検証対象外（参照素材として凍結）。
