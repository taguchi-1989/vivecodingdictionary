作成日: 2026-05-28

## タスクの意図

`assets/ponchi/final` の本番ポンチ絵を、誌面プレビューで扱いやすい **2:1 横長画像**にそろえる。同時に、画像生成ポリシーに反する画像を機械的なリサイズで済ませず、再生成または公式素材合成を前提に差し替える。

この作業は「正方形画像を横長キャンバスへ置く」だけでは終わらない。商用配布を想定するため、ロゴ、公式アイコン、公式マーク、ブランド色、読める文字、実在 UI、実在人物の顔、白黒青以外の色を含む画像は、2:1 化の前に採用可否を判断する。

## ポリシー準拠の採用基準

OK 条件:

- 2:1 の WebP として保存されている。
- 白、黒、グレー、濃紺、薄青だけで成立している。
- 背景は白または極薄青。暗い背景や装飾背景ではない。
- 読める文字が画像内にない。必要な文字は本文・キャプション側に置く。
- 会社ロゴ、サービスロゴ、公式アイコン、公式マーク、実在 UI を AI 生成で再現していない。
- 実在人物の肖像に寄せていない。
- 200px 程度に縮小しても主題が分かる。

NG または再生成候補:

- ブランドカラー、虹色、黄、緑、赤、紫、茶、オレンジが目立つ。
- ロゴ風・公式アイコン風・公式マーク風の図形が AI 生成で混ざっている。
- 画面内の文字が読める、または崩れた文字に見える。
- 1:1 の中央配置では意味が弱く、横長構図として作り直す必要がある。
- 既存シリーズと線の密度やキャラ造形が大きく違う。
